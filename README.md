Lista inlantuita
	Este o structură de date în care elementele (obiectele) sunt aranjate într-o anumită ordine. Aceasta se deosebește de un tablou prin faptul că ordinea este determinată de pointerii (legăturile) conținuți în fiecare element, nu de indici. Listele pot fi simplu înlănțuite sau dublu înlănțuite.
O listă dublu înlănțuită conține elemente care au un câmp cheie și două câmpuri pointer: unul către următorul element și unul către precedentul.
Dacă lista este simplu înlănțuită, fiecare element conține doar câmpul cheie și un pointer către următorul element din listă.
Santinela este un obiect fictiv utilizat pentru a simplifica tratamentul condițiilor de la extremitățile listei. Astfel, pointerul către elementul precedent al capului listei va indica spre santinelă (NIL), iar pointerul următorului element al cozii va indica, de asemenea, spre acest obiect. O listă fără santinelă nu are nod fictiv, iar operațiile de inserare și ștergere trebuie să trateze separat cazurile de la începutul și sfârșitul listei. În schimb, o listă cu santinelă folosește un nod fictiv care marchează capul și coada, simplificând implementarea și eliminând condițiile speciale, în schimbul unui consum minim suplimentar de memorie.
